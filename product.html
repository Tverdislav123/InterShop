Лучше не пиши на чистом пхп, а возьми какой-нибудь простенький фреймворк. Тот же slim или silex. По крайней мере, они дадут тебе знания о том, как использовать автозагрузку и разделение на код и темплейт.

Поставь нормальную IDE. К примеру, PHPStorm. И выставь там в настройках Editor -> Code Style -> PHP «Set from...» -> PSR1/PSR2. Для базы данных можешь использовать либо чистый PDO, либо взять Doctrine DBAL.

Есть еще Doctrine ORM, но это сложно, пожалуй. А DBAL позволяет выполнять запросы в базу, делаю привязку параметров. Также из микрофреймворков есть Lumen.
